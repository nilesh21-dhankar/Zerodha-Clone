##  Zerodha Clone (Basic Full Stack Project)

This is a *basic full stack project inspired by Zerodha*.  
The main goal of this project was to understand how *frontend and backend work together*.

### Technologies Used
- React
- Node.js
- Express.js
- MongoDB

### What I worked on
- Basic UI for frontend
- Backend APIs
- Connecting frontend with backend
- Storing data in database
